### B4.1 MarkText — 高颜值开源 Markdown 编辑器

- **链接**：https://mp.weixin.qq.com/s/tEsdMz7irrwnB4hi9GAdcg
- **类型**：公众号文章（小 G 小 G / GitCake）
- **访问状态**：✅ 正常
- **主题**：推荐开源 Markdown 编辑器 MarkText

#### 核心观点
- MarkText 是 Typora 的免费开源替代品，提供所见即所得的 Markdown 写作体验
- 三种编辑模式：打字机模式（当前行居中）、专注模式（非编辑段落变灰）、源码模式
- 六套主题（三浅色 + 三深色）、表格拖拽编辑、KaTeX 公式、导出 HTML/PDF

#### 关键信息
- GitHub 仓库：https://github.com/marktext/marktext
- 跨平台：Windows/macOS/Linux
- macOS 安装：`brew install --cask mark-text`

#### 内容性质判断
- 工具推荐 — 轻量实用型内容

#### 信息密度
- **低** — 功能列举为主，没有深度对比或使用技巧

#### 适合谁读
- 在找免费 Markdown 编辑器替代 Typora 的人

#### 我的判断
- **最值得带走的结论**：如果你在用付费 Typora 且只需要基础 Markdown 编辑，MarkText 是免费替代选项
- **噪音部分**：文章偏营销向，实际功能描述不深

#### 建议的后续行动
- 如果有 Markdown 编辑器需求，brew 安装试用 5 分钟即可判断是否满足需要
- 对比 Obsidian、Zed 等更强大的编辑器再做决定

## B7. 其他

### B7.1 Hive Minds — Peter Watts 科幻小说虚构文档

- **链接**：https://www.rifters.com/echopraxia/hiveminds.htm
- **类型**：科幻文学 / 虚构文档
- **访问状态**：✅ 正常

#### 主要用途
- Peter Watts 科幻小说 *Echopraxia* 的补充世界设定文档，以虚构军事技术期刊文章形式描述蜂群意识（hive mind）

#### 核心内容
- **Moksha Mind**：大规模宗教性蜂群网络，追求自我消解，对冲突不感兴趣；依赖电子通信，易被 EMP 和信号干扰瘫痪
- **Bicameral Order**：更小、更危险的蜂群——有机神经网络连接，免疫信号干扰；保持清醒的集体自我意识；近年产出大量颠覆性科技突破
- 虚构文档以军事威胁评估的口吻，分析两种蜂群意识的作战能力和地缘风险
- Bicameral 修道院正在获取工业级能源输出能力，用途不明

#### 研究价值
- 对 AI agent 多体协作、集群智能的隐喻思考有启发
- Peter Watts 是硬科幻作家，其对集群意识的想象有神经科学基础
- 可以作为思考"AI 蜂群 vs 人类组织"的文学参照

#### 我的判断
- 这是科幻世界设定文档，不是技术资料——但 Watts 对有机蜂群（免疫 EMP）vs 电子蜂群（易被干扰）的对比框架，对思考 AI agent 网络的脆弱性有类比价值
- Echopraxia 本身是一本值得一读的硬科幻小说

#### 建议的后续行动
- 如果对集群智能的科幻想象感兴趣，读 Watts 的 *Blindsight* → *Echopraxia* 两部曲
- 作为 AI agent 协作的思维拓展素材收藏
